## About

Python is a high-level programming language designed for readability and simplicity.
It has a clean syntax and a large standard library which makes it one of the most popular languages
for web development, data science, automation, and scripting.

## Key Features

> - Interpreted, runs line by line without a compile step
> - Huge ecosystem (PyPI hosts 500k+ packages)
> - Strong support for OOP, functional, and procedural styles
> - Used widely in AI/ML (PyTorch, TensorFlow, scikit-learn)
> - Great for beginners and experts alike

## History - Timeline

> - **1980s**: Conceptualized as a successor to the ABC language  
> - **1989**: Development started by Guido van Rossum at CWI in the Netherlands
> - **1991**: First released by Guido van Rossum  
> - **1994**: Python 1.0 released with new features like lambda, map, filter, and reduce
> - **2000**: Python 2  released with new features but also some backward-incompatible changes
> - **2008**: Python 3 released to address design flaws and modernize the language  
> - Today: Python is one of the **most popular programming languages** in the world, used by companies like Google, Instagram, and NASA.

## History

Python was thought of in the late 1980s by Guido van Rossum
at Centrum Wiskunde & Informatica (CWI) in the Netherlands.
It was designed as a successor to the ABC programming language.
Python implementation began in December of 1989.
Van Rossum first released it in 1991. He assumed sole responsibility for the project,
as the lead developer, until July 12 2018.
In January 2019, active Python core developers elected a five-member Steering Council to lead the project.

### Fun Fact:

The name Python comes from the British comedy series Monty Python's Flying Circus

## Syntax

Python uses whitespaces for its indentation instead of curly brackets or keywords.
Increase in indentation comes after certain statements like `if`, `for`, `while`, `def`, and `class`.
Therefore the languages visual structure is very easy and clear to understand.

## Statements

Statements in Python include:

> - `if` statements for conditional execution
> - `else` and `elif` for additional conditions
> - `for` and `while` loops for iteration
> - `try` and `except` for error handling
> - `raise` for throwing exceptions
> - `class` for defining classes
> - `def` for defining functions
> - `with` for context management
> - `break`, `continue`, and `pass` for controlling loop flow
> - `del` for deleting variables or objects
> - `assert` for debugging and testing
> - `yield` for generator functions
> - `return` for returning values from functions
> - `import` for importing modules and packages
> - `match` and `case` for structural pattern matching

## Expressions

Expressions in Python include:

> - `+`, `-`, `*` for mathematical addition, subtraction and multiplication
> - `/`, `//`, `**` for division, floor division and exponentiation
> - `+` for string concatenation, `*` for string repetition
> - `@` for matrix multiplication
> - `:=`, the walrus operator for assignment expressions
> - `==`, `!=`, `<`, `>`, `<=`, `>=` for comparison
> - `and`, `or`, `not` for logical operations
> - `in`, `not in` for membership testing
> - `is`, `is not` for identity testing
> - `[]`, `()`, `{}` for list, tuple, and dictionary literals
> - `lambda` for anonymous functions
> - `f"..."` for formatted string literals (f-strings)
